# WebFishing Dedicated Server

This is a dedicated server for the game WebFishing.  
It is a simple server that can be used to host a game of WebFishing.

## Supported Version
- 1.09

## Configuration
modify appsettings.json file

### Room Types
- Public
- CodeOnly
- FriendsOnly
- Private

### Room Code
- 6 characters (A-Z, 0-9)
